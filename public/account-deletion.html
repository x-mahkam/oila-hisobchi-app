HISOBNI O‘CHIRISH SIYOSATI

Oxirgi yangilangan sana: 13.07.2026

1. Umumiy qoidalar

"Oila Hisobchi" ilovasi foydalanuvchilarga o‘z hisoblarini istalgan vaqtda butunlay o‘chirish imkoniyatini taqdim etadi.

Hisob o‘chirilgandan so‘ng foydalanuvchining barcha shaxsiy ma'lumotlari va ilovadagi faoliyatiga oid ma'lumotlar qayta tiklab bo‘lmaydigan tarzda o‘chiriladi.

2. Hisobni qanday o‘chirish mumkin?

Hisobni quyidagi tartibda o‘chirish mumkin:

Profil → Sozlamalar → Hisobni o‘chirish

Hisobni o‘chirishdan oldin:

  • foydalanuvchi parolini qayta kiritishi;
  • o‘chirishni tasdiqlashi talab etiladi.

Bu foydalanuvchi xavfsizligini ta'minlash uchun amalga oshiriladi.

3. O‘chiriladigan ma'lumotlar

Hisob o‘chirilganda quyidagi ma'lumotlar butunlay o‘chiriladi:

  • Profil ma'lumotlari
  • Oilaviy ma'lumotlar
  • Daromadlar
  • Xarajatlar
  • Qarzlar
  • Maqsadlar
  • Statistikalar
  • Coin ma'lumotlari
  • Orzular (Wish)
  • Vazifalar
  • Rag'bat pullari
  • Bildirishnomalar
  • Firebase Authentication hisob ma'lumoti
  • Firestore ma'lumotlari
  • Firebase Storage fayllari
  • FCM tokenlari

4. Bola akkauntlari

Bola akkauntlari mustaqil ravishda o‘z hisoblarini o‘chira olmaydi.

Faqat ota-ona (asosiy oila administratori) quyidagi huquqlarga ega:

  • bola akkauntini o‘chirish;
  • bola ma'lumotlarini boshqarish;
  • bola akkauntini oiladan chiqarish.

Agar ota-ona butun oilaviy hisobni o‘chirsa, barcha bog‘langan bola akkauntlari ham birgalikda o‘chiriladi.

5. Qayta tiklash

Hisob o‘chirilgandan keyin ma'lumotlarni qayta tiklashning imkoni ma'vjud emas.

Shu sababli foydalanuvchidan hisobni o‘chirishdan oldin qarorini diqqat bilan ko‘rib chiqishi tavsiya etiladi.

ACCOUNT DELETION POLICY

Last updated: 13.07.2026

1. General Provisions

The "Oila Hisobchi" app provides users with the option to completely delete their accounts at any time.

Once the account is deleted, all personal data of the user and data related to their activity in the app are permanently and irreversibly deleted.

2. How to delete an account?

An account can be deleted in the following way:

Profile → Settings → Delete Account

Before deleting:

  • the user must re-enter their password;
  • confirm the deletion.

This is done to ensure user security.

3. Deleted Data

When an account is deleted, the following data is completely wiped:

  • Profile details
  • Family details
  • Income
  • Expenses
  • Debts
  • Goals
  • Statistics
  • Coin data
  • Wishes (Wish)
  • Tasks
  • Incentives / Allowance
  • Notifications
  • Firebase Authentication credentials
  • Firestore data
  • Firebase Storage files
  • FCM tokens

4. Child Accounts

Child accounts cannot independently delete their own accounts.

Only the parent (the main family administrator) has the following rights:

  • delete a child account;
  • manage child data;
  • remove a child account from the family.

If the parent deletes the entire family account, all linked child accounts will be deleted as well.

5. Recovery

Once an account is deleted, it is impossible to recover any data.

Therefore, users are advised to carefully consider their decision before deleting the account.

6. Contact Us

If you have any questions, you can contact us via the "Contact Us" section inside the app or through the developer's email address:
Email: oilahisobchi@gmail.com

ПОЛИТИКА УДАЛЕНИЯ АККАУНТА

Дата последнего обновления: 13.07.2026

1. Общие положения

Приложение «Oila Hisobchi» предоставляет пользователям возможность полностью удалить свои аккаунты в любое время.

После удаления аккаунта все персональные данные пользователя и информация о его активности в приложении удаляются безвозвратно.

2. Как удалить аккаунт?

Удалить аккаунт можно следующим образом:

Профиль → Настройки → Удалить аккаунт

Перед удалением аккаунта требуется:

  • повторно ввести пароль пользователя;
  • подтвердить удаление.

Это делается для обеспечения безопасности пользователя.

3. Удаляемые данные

При удалении аккаунта полностью удаляются следующие данные:

  • Данные профиля
  • Семейные данные
  • Доходы
  • Расходы
  • Долги
  • Цели
  • Статистика
  • Данные о коинах
  • Желания (Wish)
  • Задачи
  • Поощрительные средства
  • Уведомления
  • Учетные данные Firebase Authentication
  • Данные Firestore
  • Файлы Firebase Storage
  • Токены FCM

4. Детские аккаунты

Детские аккаунты не могут самостоятельно удалить свои профили.

Только родитель (главный администратор семьи) имеет следующие права:

  • удалить детский аккаунт;
  • управлять детскими данными;
  • исключить детский аккаунт из семьи.

Если родитель удаляет весь семейный аккаунт, все связанные детские аккаунты также удаляются.

5. Восстановление данных

После удаления аккаунта восстановление данных невозможно.

По этой причине пользователям рекомендуется внимательно обдумать свое решение перед удалением аккаунта.

6. Свяжитесь с нами

Если у вас возникнут вопросы, вы можете обратиться через раздел «Связаться с нами» в приложении или по электронной почте разработчика:
